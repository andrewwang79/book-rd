# 团队结构

## 规划
* 基础岗位是必须的，开发小组数量可随着开发工作量增减

### 基础岗位
建议4人

| 岗位 | 数量 | 必须条件 |
| :----: | ---- | ---- |
| 技术总监 | 1 | Y |
| 技术经理/架构师 | 1 | 产品复杂度(业务/技术) |
| 运维工程师 | 1 | 自有机房 |
| iOS软件工程师 | 1 | 开发app |
| Android软件工程师 | 1 | 开发app |

### 开发小组
默认8人，一般6-8人

| 岗位 | 数量 | 必须 |
| :----: | ---- | ---- |
| 产品经理 | 1 | Y |
| 网页交互设计师 | 1 | Y |
| 测试工程师 | 1 | Y |
| 高级软件工程师 | 1 | Y |
| 中级软件工程师 | 1 | Y |
| 初级软件工程师 | 1 | 开发工作量 |
| 高级前端工程师 | 1 | Y |
| 中级前端工程师 | 1 | Y |
