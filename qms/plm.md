# 产品生命周期
## 阶段
* 从研发到退市的完整阶段

| 阶段 | 通用名称 | 医疗器械对应名称 | 活动和产出 |
| - | - | - | - |
| 1 | 需求/机会 | 立项 | 商业立项、项目立项，用户需求 |
| 1 | 概念 | 策划 | 计划 |
| 2 | 设计 | 设计输入 | 产品需求(软硬件的功能和指标)，系统需求 |
| 3 | 设计 | 设计输出转换 | 系统架构设计，软硬件的系统设计 <br/> 开发，集成测试 |
| 4 | 验证 | 设计验证 | 系统测试、版本发布 <br/> **型检**、补检 |
| 5 | 验证 | 设计确认 | **临床(试验/评价)**，注册临床，产品发布 |
| 6 | 生产 | 设计转移 | 注册许可证 |
| 7 | 上市 | 上市 | 生产许可证，项目结项 |
| 8 | 上市后监督 | 上市后监督(PMS) | 量产、产品维护 <br/> 客户投诉、不良事件 <br/> 飞检、国抽 |
| 9 | 退市 | 退市 |  |
