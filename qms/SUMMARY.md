# 质量管理体系(QMS)
* 质量不是检测出来的，是通过质量管理体系生产出来的
* QMS的文件体系，第一层是金字塔顶

| 层 | 名称 | 说明 |
| :-: | - | - |
| 第一层 | 质量手册 | 公司层面的指导。含方针、要求等 |
| 第二层 | 程序文件 | 程序说明，不涉及具体操作 |
| 第三层 | 具体操作流程 |  |
| 第四层 | 记录文件 | 活动流程的过程记录 |

## 流程
* 质量控制(QC)
* 质量保证(QA)
* 质量验证(VA)

## 风险评价准则
| 概率/严重程度	| 灾难性 | 致命 | 中度 | 轻度 |
| - | - | - | - | - |
| 经常 | U | U | U | R |
| 有时 | U | U | R | R |
| 偶然 | U | R | R | R |
| 很少 | R | R | R | A |
| 非常少 | R | R | A | A |
| 极少 | R | A | A | A |

>说明：
A：可接受的风险
R：合理可行降低(ALARP)的风险
U：不经过风险／ 收益分析即判定为不可接受的风险
