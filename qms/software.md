# 软件

## 应用类型
* 系统软件、应用软件、中间件

## SOUP
* 不确定谱系软件/未知来源软件(SOUP/Software of Unknown Provenance)
* 生产企业**未进行完整生命周期管理**的软件。已经开发且通常可得到的，但来源、可靠性或安全性未知。开发团队需对其进行评估和管理，确保可靠性和安全性。

## 基于PLM的分类
| 项	| 英语 | 定义 | 实例 | 属于SOUP |
| - | - | - | - | - |
| 遗留软件 | Legacy Software | 企业以前开发，现在不能得到足够开发记录的软件组件或工具 | 验证工具 | Y |
| 外包软件 |  | 企业委托第三方开发的定制软件组件或工具 |  | Y |
| 现成软件 | OTS(Off-the-shelf Software) | 开源/其他企业开发的软件组件或工具 | 开源库(如Boost)、中间件(如MySQL)、应用软件(如机械臂驱动配置软件) | Y |
| 成品软件 | Custom Software | 生产企业开发的独立的最终产品，使用方是最终用户 <br> 非成品软件可以被集成到成品软件中，如Boost和机械臂驱动配置软件 | 具体产品 | N |

## 质量管理软件
> 质量管理所用的应用软件。非成品软件，没有时成品软件也可安装运行

| 用途	| 范围 | 实例 | 说明 |
| - | - | - | - |
| 配置管理 | 文件管理、配置管理 | PLM |  |
| 设计开发 | 开发，测试，代码管理，缺陷管理，研发管理 | VsCode、Git、Jira |  |
| 生产 | 物料管理，生产管理，生产软件，检验软件 | MES、刻录软件、测试软件 |  |
